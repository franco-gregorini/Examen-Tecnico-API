# Examen-Tecnico-API
Se comparte el proyecto de automatización de API usando el Framework Karate

## Requisitos
- **Java 21** o superior
- **Maven 3.6+**
- **Git**
- **Cucumber** (Instalarlo como extension de VSC o su herramienta)

## Instalación
Clona el repositorio:
- git clone (https://github.com/franco-gregorini/Examen-Tecnico-API.git)

## Ejecución
En el terminal ejecuta el código
- mvn -Dtest=Runner test

Para ver los reportes:
- Entra a target/karate-reports y abrir el archivo karate-summary.html en tu navegador
